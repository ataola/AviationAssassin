# 中国低成本航空行李规则专题页

> 看懂规则，不做低价票补价受害者。

一组静态 `HTML` 页面，整理中国境内仍在运营、且官网仍能核验低成本/差异化行李售卖逻辑的航空公司行李规则，重点服务于这两个场景：

- 买最低价票前，快速判断自己会不会在行李上被补价
- 把复杂规则整理成适合阅读、截图、打印和转发的页面

## 文件结构

- `china-lcc-baggage-report.html`
  完整报告页，包含执行摘要、风险矩阵、天津航空单独聚焦、逐家规则摘要与来源说明。
- `lowest-fare-baggage-cheatsheet.html`
  极简速查页，适合买票前 30 秒快速扫一眼。
- `china-lcc-baggage-poster.html`
  单页海报版，适合截图传播、打印或发群。

## 收录范围

本专题采用“保守可验证口径”：

- 只收目前仍在中国境内实际运营的相关航司
- 只收官网仍能核出低成本/差异化行李售卖逻辑的样本
- 不硬塞运营状态不清、历史上被称为廉航但当前官网规则难核实的对象

当前专题样本：

- 春秋航空
- 中国联合航空
- 天津航空
- 西部航空
- 祥鹏航空
- 长安航空
- 九元航空

## 页面特点

- 纯静态 `HTML/CSS/SVG`，无外部依赖，双击即可本地打开
- 专门区分“小尺寸随身包”和“20 寸登机箱”两类常见误判
- 突出最低价票最容易踩坑的 3 类问题：
  - 默认 `0kg` 托运
  - 免费手提不等于可带 `20 寸箱`
  - 登机口/机场现场补价明显更贵
- 天津航空单独展开，便于聚焦查看

## 使用建议

- 想系统阅读：打开 `china-lcc-baggage-report.html`
- 想快速决策：打开 `lowest-fare-baggage-cheatsheet.html`
- 想发给别人或截图：打开 `china-lcc-baggage-poster.html`
- 想统一入口：打开 `index.html`

## 数据与方法说明

- 优先依据航司官网、官网公开规则页、官网 PDF、官网关联公开页面
- 能静态核实的数字尽量写清到 `元/件`、`元/kg`、`kg`、`尺寸`
- 对于依赖实时页面、按航线/舱位动态变化的价格，不硬写统一价目表，而明确标注“以下单页实时展示为准”
